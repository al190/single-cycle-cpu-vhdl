# Single Cycle CPU in VHDL

This project is a VHDL implementation of a single-cycle CPU developed in Xilinx Vivado.  
It includes the main datapath and control components required for CPU operation, along with testbenches for simulation and verification.

## Features
- Program Counter
- ALU
- Register File
- Memory
- Control logic
- Testbenches for verification

## Tools Used
- VHDL
- Xilinx Vivado

## Project Structure
- `src/` → source VHDL files
- `tb/` → testbench files
- `constraints/` → FPGA constraint files
- `docs/` → notes and documentation

## Goal
The goal of this project is to design and verify a simple single-cycle CPU architecture for educational and FPGA development purposes.
